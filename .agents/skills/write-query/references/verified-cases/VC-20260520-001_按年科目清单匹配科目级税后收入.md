# 按年科目清单匹配科目级税后收入

来源：write-query 交互验证（政企科目维表 + 三年分列匹配）

**定位**：复杂场景 **CTAS 过程表流水线** 标准模板（非 `WITH` 默认编排）。

## 适用

- 用户按 **外部科目维表**（`due_income_code23/24/25` 分列）匹配 **SR 科目税后收入**。
- 账期按 **年** 汇总（`substr(par_month_id,1,4)`），输出局向、营服、产权客户、产品、揽装人等维度 + `sum(fee_all)`。
- 用户采用 **按年建临时表 + 按年 JOIN 维表列** 的生产写法；需求方需对中间表单独改过滤/验收。

## 不适用

- 服务级划小收入汇总（无科目维度）→ 047 最终版划小收入、`sum(a0)`。
- 入网、到达、积分、续约类需求。

## 主表与口径

- **主表**：048 全量科目级收入 `dwm_srhx_src_income_list_mon`（或分局视图 `zone_gz.view_{分局}_ads_srhx_src_income_list_mon`）。
- **粒度**：号码/服务级明细（每行一个 `fee_all`）；汇总时对输出维度 `sum(fee_all)`。
- **度量**：`sum(fee_all)`（税后收入）；用户口头「确认收入（税后）」在本场景对齐 `fee_all`。
- **分区**：`par_month_id`（月，如 `202301`）；账期年由 `substr(cast(par_month_id as string), 1, 4)` 派生。
- **常用过滤（口径确认阶段选定，非技能全局默认）**：`contract_flag = 1`（划小）、`is_filter = 0`（非主营科目，参考 `M-TOPIC-REV-006`）。用户历史脚本未加时，在「业务口径确认」列出待确认项，不要 silent 改数。
- **营服**：默认 **划小营服** `branch_name`，不用 `channel_branch_name`（揽装营服）。

## 字段映射

| 需求字段 | 来源 |
|----------|------|
| 账期（年） | `substr(cast(par_month_id as string), 1, 4)` 或汇总层 `year_id` |
| 划小局向 | `subst_name` |
| 所属营服 | `branch_name` |
| 产权客户名称 | `cust_name` |
| 产权客户编码 | `cust_nbr` |
| 产品名称 | `prod_name` |
| 揽装人 | `sales_name` |
| 揽装人编码 | `sales_code` |
| 确认收入（税后） | `sum(fee_all)` |

## 补表

- **科目维表**（用户自建）：`ads_yz_zqb_xc_due_income_code_dim_list`
  - `due_income_code23` / `due_income_code24` / `due_income_code25` 分别对应 2023/2024/2025 年 SR 科目白名单。
  - 按年 JOIN：23 年 `a.due_income_code = b.due_income_code23` 且 `par_month_id` 当年 1–12 月；24/25 年同理。
  - 维表多行同码会放大明细行；按年拆临时表可降低风险并便于分段改数。

## 过程表血缘

| 步骤 | 表名 | 用途 | 上游 | 关键过滤 | 验收 SQL |
|------|------|------|------|----------|----------|
| 1 | `ads_yz_zqb_xc_due_income_code_dim_list` | 科目维表（三列年码） | 配置表 `zone_gz_yz_3461990841409536` | - | `select count(*) from ads_yz_zqb_xc_due_income_code_dim_list` |
| 2 | `tmp_ads_yz_zqb_xc_code_cust_sr_dwb_23` | 2023 年科目匹配明细 | 048 + 维表 | `par_month_id` 202301–202312；JOIN `due_income_code23` | `select count(*) from tmp_ads_yz_zqb_xc_code_cust_sr_dwb_23` |
| 3 | `tmp_ads_yz_zqb_xc_code_cust_sr_dwb_24` | 2024 年明细 | 同上 | 202401–202412；`due_income_code24` | `select count(*) from tmp_ads_yz_zqb_xc_code_cust_sr_dwb_24` |
| 4 | `tmp_ads_yz_zqb_xc_code_cust_sr_dwb_25` | 2025 年明细 | 同上 | 202501–202512；`due_income_code25` | `select count(*) from tmp_ads_yz_zqb_xc_code_cust_sr_dwb_25` |
| 5 | `tmp_ads_yz_zqb_xc_code_cust_sr_dwb_union` | 三年明细合并 | 步骤 2–4 | `union all` | `select count(*) from tmp_ads_yz_zqb_xc_code_cust_sr_dwb_union` |
| 6 | `ads_yz_zqb_xc_code_cust_sr_dwb` | 按年+维度汇总结果 | union 表 | `group by` + `sum(fee_all)` | `select count(*) from ads_yz_zqb_xc_code_cust_sr_dwb` |

## SQL 编排

### 推荐模式：按年 CTAS + UNION + 汇总 CTAS

1. 建维表（从配置表导入三列科目编码）。
2. 按年各建 `tmp_..._{yy}`：`048` INNER JOIN 维表对应年列 + `par_month_id` 当年范围。
3. `tmp_..._union`：`union all` 各年临时表。
4. `ads_...`：对 union 表 `group by` 账期年 + 输出维度 + `sum(fee_all)`。

### 汇总注意

- 临时表为 **号码级** 时，内层可不 sum，但 **最终交付表** 必须有 `group by 维度 + sum(fee_all)`。
- 内外都不 sum 会同一维度多行，不是最终汇总结果。

## 参考 SQL（完整五步）

以下示例含 `contract_flag`/`is_filter`；若口径确认未选，删除对应 AND 条件。

```sql
-- 步骤1：导入科目叶子节点
drop table if exists ads_yz_zqb_xc_due_income_code_dim_list purge;
create table ads_yz_zqb_xc_due_income_code_dim_list
row format delimited fields terminated by '\u0001'
stored as orc tblproperties('orc.compression'='snappy') as
select
    index1 as due_income_code23,
    index2 as due_income_code24,
    index3 as due_income_code25
from zone_gz_yz_3461990841409536;
-- select count(*) from ads_yz_zqb_xc_due_income_code_dim_list;

-- 步骤2：23年
drop table if exists tmp_ads_yz_zqb_xc_code_cust_sr_dwb_23 purge;
create table tmp_ads_yz_zqb_xc_code_cust_sr_dwb_23
row format delimited fields terminated by '\u0001'
stored as orc tblproperties('orc.compression'='snappy') as
select a.*
from dwm_srhx_src_income_list_mon a
join ads_yz_zqb_xc_due_income_code_dim_list b
  on a.due_income_code = b.due_income_code23
where a.par_month_id >= 202301
  and a.par_month_id <= 202312
  and a.contract_flag = 1
  and a.is_filter = 0;
-- select count(*) from tmp_ads_yz_zqb_xc_code_cust_sr_dwb_23;

-- 步骤3：24年
drop table if exists tmp_ads_yz_zqb_xc_code_cust_sr_dwb_24 purge;
create table tmp_ads_yz_zqb_xc_code_cust_sr_dwb_24
row format delimited fields terminated by '\u0001'
stored as orc tblproperties('orc.compression'='snappy') as
select a.*
from dwm_srhx_src_income_list_mon a
join ads_yz_zqb_xc_due_income_code_dim_list b
  on a.due_income_code = b.due_income_code24
where a.par_month_id >= 202401
  and a.par_month_id <= 202412
  and a.contract_flag = 1
  and a.is_filter = 0;
-- select count(*) from tmp_ads_yz_zqb_xc_code_cust_sr_dwb_24;

-- 步骤4：25年
drop table if exists tmp_ads_yz_zqb_xc_code_cust_sr_dwb_25 purge;
create table tmp_ads_yz_zqb_xc_code_cust_sr_dwb_25
row format delimited fields terminated by '\u0001'
stored as orc tblproperties('orc.compression'='snappy') as
select a.*
from dwm_srhx_src_income_list_mon a
join ads_yz_zqb_xc_due_income_code_dim_list b
  on a.due_income_code = b.due_income_code25
where a.par_month_id >= 202501
  and a.par_month_id <= 202512
  and a.contract_flag = 1
  and a.is_filter = 0;
-- select count(*) from tmp_ads_yz_zqb_xc_code_cust_sr_dwb_25;

-- 步骤5：合并
drop table if exists tmp_ads_yz_zqb_xc_code_cust_sr_dwb_union purge;
create table tmp_ads_yz_zqb_xc_code_cust_sr_dwb_union
row format delimited fields terminated by '\u0001'
stored as orc tblproperties('orc.compression'='snappy') as
select * from tmp_ads_yz_zqb_xc_code_cust_sr_dwb_23
union all
select * from tmp_ads_yz_zqb_xc_code_cust_sr_dwb_24
union all
select * from tmp_ads_yz_zqb_xc_code_cust_sr_dwb_25;
-- select count(*) from tmp_ads_yz_zqb_xc_code_cust_sr_dwb_union;

-- 步骤6：统计
drop table if exists ads_yz_zqb_xc_code_cust_sr_dwb purge;
create table ads_yz_zqb_xc_code_cust_sr_dwb
row format delimited fields terminated by '\u0001'
stored as orc tblproperties('orc.compression'='snappy') as
select
    200 as city_id,
    substr(cast(par_month_id as string), 1, 4) as year_id,
    subst_id,
    subst_name,
    branch_id,
    branch_name,
    cust_nbr,
    cust_name,
    prod_id,
    prod_name,
    sales_id,
    sales_code,
    sales_name,
    contract_flag,
    is_filter,
    sum(fee_all) as sh
from tmp_ads_yz_zqb_xc_code_cust_sr_dwb_union
group by
    substr(cast(par_month_id as string), 1, 4),
    subst_id,
    subst_name,
    branch_id,
    branch_name,
    cust_nbr,
    cust_name,
    prod_id,
    prod_name,
    sales_id,
    sales_code,
    sales_name,
    contract_flag,
    is_filter;
-- select count(*) from ads_yz_zqb_xc_code_cust_sr_dwb;
-- select * from ads_yz_zqb_xc_code_cust_sr_dwb;
```
