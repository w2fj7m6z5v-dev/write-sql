# 小业务 T+1 活跃率

来源：`CDAP自助分析常用统计语句分享.docx`

## 适用

- 用户问存量融合宽带里 T+0 月新增小业务在 T+1 月仍活跃/在网占比。
- 用户要按视联网、企业云盘、商企云电脑、FTTR、ITV、天翼看家等小业务分项统计。

## 不适用

- 普通小业务入网量、到达量。

## 主表与口径

- 使用小业务融合质态多维表。
- 天翼云盘、ITV、天翼看家按省活跃口径判断有效。
- 其它业务按当月/下月是否在网判断。

## 字段映射

| 需求字段 | 来源 |
|----------|------|
| 分子 | `*_fenzi_1` |
| 分母 | `*_fenzi_2` |
| 活跃率 | `sum(*_fenzi_1)/sum(*_fenzi_2)` |

## 补表

默认不补表。

## SQL 编排

- 按 `xyw_name` 分业务统计分子和分母。
- 视联网 = 视联网监控设备 + 视联网 AI 单品。
- 有重复风险的小业务新增量，按维度使用 `gz_rank`、`subst_rank`、`branch_rank`、`area_rank` 去重；不同维度汇总不可简单相加。
