# 转化率剔除撤单重受理

来源：`CDAP自助分析常用统计语句分享.docx`

## 适用

- 用户问宽带新装或移机转化率，且要求全局剔除撤单重受理。

## 不适用

- 普通入网量、到达量；这些默认走 069。

## 主表与口径

- 主表：060 转化率。
- 受理时间：`act_date`。
- 状态时间：`subs_stat_date`。
- 剔除撤单重受理：`is_cd_act=0`。

## 字段映射

| 需求字段 | 来源 |
|----------|------|
| 撤销 | `hj_zh='撤销'` |
| 缓装 | `hj_zh='缓装'` |
| 开通中 | `hj_zh='开通中'` |
| 完工 | `hj_zh='完工'` |
| 订单量 | `subs_code` |

## 补表

默认不补表。

## SQL 编排

```sql
SELECT
  bd_std_subst_name,
  COUNT(CASE WHEN prod_type_1='普通宽带' AND action_type_desc='新装' AND hj_zh='撤销' AND is_cd_act=0 THEN subs_code END) AS cancel_cnt,
  COUNT(CASE WHEN prod_type_1='普通宽带' AND action_type_desc='新装' AND hj_zh='缓装' AND is_cd_act=0 THEN subs_code END) AS slow_cnt,
  COUNT(CASE WHEN prod_type_1='普通宽带' AND action_type_desc='新装' AND hj_zh='开通中' AND is_cd_act=0 THEN subs_code END) AS opening_cnt,
  COUNT(CASE WHEN prod_type_1='普通宽带' AND action_type_desc='新装' AND hj_zh='完工' AND is_cd_act=0 THEN subs_code END) AS finish_cnt,
  COUNT(CASE WHEN prod_type_1='普通宽带' AND action_type_desc='新装' AND is_cd_act=0 THEN subs_code END) AS total_cnt
FROM view_ads_yz_zhl_list
WHERE date_format(act_date,'yyyyMMdd') BETWEEN '${start_day}' AND '${end_day}'
  AND date_format(subs_stat_date,'yyyyMMdd') BETWEEN '${start_day}' AND '${finish_end_day}'
GROUP BY bd_std_subst_name
;
```
