# 收入多维查询模板

来源：`CDAP自助分析常用统计语句分享.docx`

## 适用

- 用户问按分局、营服、包区、产品统计划小收入、产数收入、基本面收入。
- 用户问按科目、收入来源统计税前/税后收入。

## 不适用

- 入网、到达、积分、续约类需求。

## 主表与口径

- 划小/基本面/产数收入：优先收入服务级多维表。
- 科目收入：优先全量科目级收入表。
- 月份：`par_month_id=${month_id}`。

## 字段映射

| 需求字段 | 来源 |
|----------|------|
| 税后收入 | `sum(a0)` 或 `sum(fee_all)` |
| 税前收入 | `sum(charge_all)` |
| 产数收入 | `sum(fee_cs)` |
| 基本面收入 | `sum(fee_fm_new)` |
| 科目 | `due_income_code`、`due_income_name` |
| 收入来源 | `data_src_type`、`data_src_name` |

## 补表

默认不补表。分局视图名需按用户权限替换。

## SQL 编排

分营服/包区查询时，常用维度包括 `branch_id`、`branch_name`、`area_id`、`area_name`、`region_type`、`prod_id`、`prod_name`、`is_std_org`、`due_type`。
