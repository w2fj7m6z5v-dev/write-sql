# 移动续约多维查询模板

来源：`CDAP自助分析常用统计语句分享.docx`

## 适用

- 用户问移动续约到期实例、已续约实例、终补/话补续约率、销售部派单续约。

## 不适用

- 新装、入网、到达、拆机。

## 主表与口径

- 主表：030 移动续约清单。
- 月份字段：`month_id`。
- 到期月份：`limit_month`。

## 字段映射

| 需求字段 | 来源 |
|----------|------|
| 到期实例数 | `count(msobjgrp_id)` |
| 已续约实例数 | `count(case when xy_state='已续约' then msobjgrp_id end)` |
| 终补到期实例数 | `disc_type_dl='终端补贴'` |
| 终补到期已续约终补实例数 | `disc_type_dl='终端补贴' and xy_state='已续约' and xy_disc_type_dl='终端补贴'` |

## 补表

默认不补表。

## SQL 编排

按 `limit_month`、`subst_name`、`branch_name`、`area_name` 分组；用户要求销售部派单时叠加 `target_channel_type='销售部派单'`。
