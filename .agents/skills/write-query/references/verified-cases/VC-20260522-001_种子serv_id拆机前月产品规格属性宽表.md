# 种子 serv_id 拆机前月产品规格 + 附属产品属性宽表

来源：write-query 交互验证（接入号种子表 + 逻辑拆机月 + 双属性月表 + 宽表）

## 适用

- 用户提供 **`serv_id` 种子清单表**（可有 `acc_nbr`，**无拆机月**），要对每个服务打标。
- 取 **拆机前一个月** 的 **产品规格属性**（105）和/或 **附属产品属性**（106）（`attr_id` → `attr_value1` + 中文名）。
- 多个 `attr_id`，输出 **宽表**：
  - 产品规格：`attr_{id}_val` / `attr_{id}_name`
  - 附属产品：`subattr_{id}_val` / `subattr_{id}_name`（前缀区分，避免 attr_id 碰撞）
- 拆机口径未特说明时，默认 **逻辑拆机**。
- **仅附属产品属性**（不要产品规格）时：可跳过 Step2a，只跑 Step2b + Step3 附属产品列。

## 不适用

- 用户明确 **物理拆机** 且有拆机订单事实 → 改 `is_wl_cancel_user=1` + `wl_cancel_subs_stat_date`。
- 取 **当前在网** 特性 → 特性/附属**日表**（`tb_pre_cm_attr_all` / `rpt_comm_cm_subserv`）。
- 销售品在档 / 订购动作 → 014 / 041，不是特性/附属产品资料表。
- 固定账期打标（种子已带拆机月且不需反查）→ 可简化 Step1，但仍用对应月表。

## 主表与口径

- **驱动表**：用户种子表（如 `tmp_acc_nbr_list`），键 `serv_id`。
- **拆机月定位**：069 **月表** `dwm_yz_tb_comm_cm_all_mon_final`；**逻辑拆机** `is_cancel_user=1`；`cancel_month_id = par_month_id`；时间 `hist_create_date`；多次拆机取最近。
- **属性月**：`attr_month_id = cancel_month_id - 1`。
- **产品规格快照（105）**：`iodata_ods_month_city.tb_pre_cm_attr_all_mon`；`par_month_id=attr_month_id` + `par_corp_id='200'` + `attr_id IN (...)`。
- **附属产品快照（106）**：`iodata_ods_month_city.rpt_comm_cm_subserv_mon`；同上分区与 JOIN 键；`serv_id + attr_id` 业务唯一。
- **特性中文**：`dws_crm_cfguse.dws_attr_value`；`attr_value1 = attr_inner_value` + `city_id='200'` + `attr_id` 对齐（105/106 通用）。

## 字段映射

| 需求字段 | 来源 |
|----------|------|
| acc_nbr, serv_id | 种子表 |
| hist_create_date, cancel_month_id, attr_month_id | 069 月表 + 计算 |
| attr_{id}_val / attr_{id}_name | 105 特性月表 + 字典 |
| subattr_{id}_val / subattr_{id}_name | 106 附属产品月表 + 字典 |
| sub_prod_id | 106（可选带出） |

## 补表

| 缺口 | 补表 | JOIN 键 |
|------|------|---------|
| 拆机月 | 069 月表 | `serv_id` + `is_cancel_user=1` |
| 拆机前月产品规格 | 105 特性月表 | `serv_id` + `par_month_id=attr_month_id` + `attr_id` |
| 拆机前月附属产品 | 106 附属产品月表 | `serv_id` + `par_month_id=attr_month_id` + `attr_id` |
| 特性中文 | 015 字典表 | `attr_id` + `attr_value1=attr_inner_value` + `city_id='200'` |

## SQL 编排

CTAS 流水线（ORC + Snappy）；种子驱动全程 LEFT JOIN。

> 示例 attr_id：产品规格 `500040209`、`500040210`；附属产品 `5000XXXXX`（**运行时替换为用户给定清单**）。

```sql
-- Step1：种子 + 逻辑拆机月（069 月表）
drop table if exists tmp_ads_yz_cj_attr_cancel_base purge;
create table tmp_ads_yz_cj_attr_cancel_base
row format delimited fields terminated by '\u0001'
stored as orc tblproperties('orc.compression'='snappy') as
select
    s.acc_nbr,
    s.serv_id,
    cm.hist_create_date,
    cm.par_month_id as cancel_month_id,
    cast(date_format(add_months(cm.hist_create_date, -1), 'yyyyMM') as int) as attr_month_id
from tmp_acc_nbr_list s
left join (
    select serv_id, par_month_id, hist_create_date,
           row_number() over (partition by serv_id order by hist_create_date desc, par_month_id desc) as rn
    from dwm_yz_tb_comm_cm_all_mon_final
    where is_cancel_user = 1
) cm on s.serv_id = cm.serv_id and cm.rn = 1;

-- Step2a：拆机前一月产品规格属性长表（105）
drop table if exists tmp_ads_yz_cj_spec_snap_base purge;
create table tmp_ads_yz_cj_spec_snap_base
row format delimited fields terminated by '\u0001'
stored as orc tblproperties('orc.compression'='snappy') as
select b.serv_id, b.attr_month_id, a.attr_id, a.attr_value1
from tmp_ads_yz_cj_attr_cancel_base b
left join iodata_ods_month_city.tb_pre_cm_attr_all_mon a
    on b.serv_id = a.serv_id
   and b.attr_month_id = a.par_month_id
   and a.par_corp_id = '200'
   and a.attr_id in ('500040209', '500040210');

-- Step2b：拆机前一月附属产品属性长表（106）
drop table if exists tmp_ads_yz_cj_subserv_snap_base purge;
create table tmp_ads_yz_cj_subserv_snap_base
row format delimited fields terminated by '\u0001'
stored as orc tblproperties('orc.compression'='snappy') as
select b.serv_id, b.attr_month_id, a.sub_prod_id, a.attr_id, a.attr_value1
from tmp_ads_yz_cj_attr_cancel_base b
left join iodata_ods_month_city.rpt_comm_cm_subserv_mon a
    on b.serv_id = a.serv_id
   and b.attr_month_id = a.par_month_id
   and a.par_corp_id = '200'
   and a.attr_id in ('5000XXXXX');

-- Step3：宽表 + 中文（产品规格 + 附属产品）
drop table if exists ads_yz_cj_attr_pre_cancel_wide purge;
create table ads_yz_cj_attr_pre_cancel_wide
row format delimited fields terminated by '\u0001'
stored as orc tblproperties('orc.compression'='snappy') as
select
    b.acc_nbr, b.serv_id, b.hist_create_date, b.cancel_month_id, b.attr_month_id,

    max(case when sp.attr_id = '500040209' then sp.attr_value1 end) as attr_500040209_val,
    max(case when sp.attr_id = '500040209' then d209.attr_value_name end) as attr_500040209_name,
    max(case when sp.attr_id = '500040210' then sp.attr_value1 end) as attr_500040210_val,
    max(case when sp.attr_id = '500040210' then d210.attr_value_name end) as attr_500040210_name,

    max(case when sb.attr_id = '5000XXXXX' then sb.attr_value1 end) as subattr_5000XXXXX_val,
    max(case when sb.attr_id = '5000XXXXX' then dsub.attr_value_name end) as subattr_5000XXXXX_name

from tmp_ads_yz_cj_attr_cancel_base b
left join tmp_ads_yz_cj_spec_snap_base sp on b.serv_id = sp.serv_id
left join tmp_ads_yz_cj_subserv_snap_base sb on b.serv_id = sb.serv_id
left join dws_crm_cfguse.dws_attr_value d209
    on d209.attr_id = '500040209' and d209.city_id = '200'
   and d209.attr_inner_value = sp.attr_value1 and sp.attr_id = '500040209'
left join dws_crm_cfguse.dws_attr_value d210
    on d210.attr_id = '500040210' and d210.city_id = '200'
   and d210.attr_inner_value = sp.attr_value1 and sp.attr_id = '500040210'
left join dws_crm_cfguse.dws_attr_value dsub
    on dsub.attr_id = '5000XXXXX' and dsub.city_id = '200'
   and dsub.attr_inner_value = sb.attr_value1 and sb.attr_id = '5000XXXXX'
group by b.acc_nbr, b.serv_id, b.hist_create_date, b.cancel_month_id, b.attr_month_id;
```

### 注意

- 069 回溯场景用 **月表**，不用日表 `dwm_yz_tb_comm_cm_all_final`。
- 105/106 历史均用 **月表**；对应日表只在网。
- 105 与 106 **勿混用**；列名用 `attr_*` vs `subattr_*` 区分。
- 新增 attr_id：Step2a/2b 扩展 `IN (...)`，Step3 增加对应 `max(case...)` 列及字典 JOIN。
- Step1 建议加 `par_month_id` 范围裁剪以控性能。

### 过程表血缘

| 步骤 | 表名 | 用途 | 验收 |
|------|------|------|------|
| 1 | `tmp_ads_yz_cj_attr_cancel_base` | 拆机月/属性月 | `select count(*) ...` |
| 2a | `tmp_ads_yz_cj_spec_snap_base` | 产品规格长表（105） | `select attr_id,count(*) group by attr_id` |
| 2b | `tmp_ads_yz_cj_subserv_snap_base` | 附属产品长表（106） | `select attr_id,count(*) group by attr_id` |
| 3 | `ads_yz_cj_attr_pre_cancel_wide` | 宽表结果 | `select count(*) ...` |
