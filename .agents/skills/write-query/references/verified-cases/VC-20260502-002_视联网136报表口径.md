# 视联网 136 报表口径

来源：`CDAP自助分析常用统计语句分享.docx`

## 适用

- 用户问视联网入网数、到达数，且口径要求对标省 136 报表。
- 用户要拆分天翼看家、天翼云眼、平安慧眼。

## 不适用

- 普通“某产品入网量/到达量”且未指定视联网 136 报表口径。

## 主表与口径

- 主表：057 视联网发展规模清单。
- 入网：`par_month_id` + `subs_stat_date` 日期范围。
- 到达：只按 `par_month_id` 取账期到达。

## 字段映射

| 需求字段 | 来源 |
|----------|------|
| 天翼看家 | `action_type='tykj_dd'` + 产品 ID / AI 属性 / offer_label |
| 天翼云眼 | `action_type='tyyy_dd'`，剔除指定 offer_code 并扣减视频会议加盟 |
| 平安慧眼 | `action_type='pahy_dd'` |

## 补表

默认不补表。

## SQL 编排

以 `metrics/专题/M-TOPIC-SMB-001_视联网入网数.md` 和 `M-TOPIC-SMB-002_视联网到达数.md` 为准；本案例只提醒 20250120 后口径中天翼看家、天翼云眼、平安慧眼三段应分开统计后合并。
