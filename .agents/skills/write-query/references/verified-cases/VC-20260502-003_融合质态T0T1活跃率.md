# 融合质态 T+0 / T+1 活跃率

来源：`CDAP自助分析常用统计语句分享.docx`

## 适用

- 用户问新融合质态、移动有效系数、托收系数、总有效率、公众/商企有效率。
- 用户指定 T+0 或 T+1。

## 不适用

- 普通移动/宽带入网、到达、收入统计。

## 主表与口径

- T+0 使用当月融合质态视图，T+1 使用次月活跃视图。
- 新宽新移：`rh_type_ykj='新宽带新移动'`。
- 剔除校园和场景宽带：`coalesce(kd_desc,'-1')<>'校园翼起来'` 且 `coalesce(prod_type3,'-1') not in ('WiFi宽带','物联宽带','快捷宽带','副宽','门禁宽带','酒店宽带')`。

## 字段映射

| 需求字段 | 来源 |
|----------|------|
| 新宽新移/存量融合宽带数 | `count(distinct serv_id)` |
| 移动有效系数 | `sum(ydyx_points)/count(distinct serv_id)` |
| 托收系数 | `sum(ts_points)/count(distinct serv_id)` |
| 总有效率 | `sum(sum_fenzi)/count(distinct serv_id)` |

## 补表

默认不补表。若用户要求县分视图名，提示需替换为对应 `view_县分_*`。

## SQL 编排

用指标分子字段求和，再除以对应分母；不要把本案例口径混入普通 069 入网/到达统计。
