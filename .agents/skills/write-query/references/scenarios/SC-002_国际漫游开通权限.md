# 国际漫游开通权限

## 适用

- 判断号码是否已开通国际漫游/国漫权限。
- 输出用户开户时间、开通国漫权限时间、G IMSI、L IMSI。
- 需要同时带出号码客户、产品、局向等属性。

## 不适用

- 漫游收入、漫游结算、漫游使用行为：不要用 114 反推。
- 普通号码 IMSI：用 `SC-005_号码清单导IMSI.md`，不要走 114。

## 主表与补表

| 角色 | 表 | 用途 |
|---|---|---|
| 号码范围 | 069 全业务资料表 | 圈定号码、补客户/产品/局向等号码属性 |
| 国漫事实 | 114 国际漫游数据表 `dws_ctg.dws_mktag_download_share_guoman_label` | 已开通国漫权限号码及相关字段 |

## 关键规则

- `069.acc_nbr = guoman.msisdn`。
- 114 的 `yyyymmdd` 是日分区/统计日，不是 069 的 `par_month_id`。
- 必须按用户指定统计日过滤 `guoman.yyyymmdd`；未指定时先确认取最新分区、区间曾开通，还是按日明细输出。
- 输出开通国漫权限时间用 `reserv2`；`reserv1` 是用户开户时间，`reserv3/reserv4` 是 G/L IMSI。
- 同一号码多日可能多行，未锁统计日会放大结果。

## 字段映射

| 需求字段 | 来源 |
|---|---|
| 号码 | 069 `acc_nbr` / 114 `msisdn` |
| 用户开户时间 | 114 `reserv1` |
| 国漫权限开通时间 | 114 `reserv2` |
| G IMSI | 114 `reserv3` |
| L IMSI | 114 `reserv4` |
| 客户/产品/局向 | 069 对应字段 |

## 风险审计

- 不要用漫游结算收入表或订单表反推是否开通国漫。
- 不要把 `reserv1` 当成国漫开通时间。
- 069 账期与 114 统计日语义不同，SQL 注释或交付说明中要写清。
