# 银行表

- **Hive 表名 / 生产现网名**：`dws_crm_cfguse.dws_tb_cm_bank`
- **用途**：按银行 ID 回填银行中文名称、银行编码等维度信息。
- **常见入口**：133 外部账户表 `acct_owner_org_branch`。

## 字段说明

| 字段 | 字段类型 | 字段含义 | 说明 |
| ------ | ------ | ---------- | ------ |
| bank_id | string | 银行标识 | 银行维度主键；常与 133.`acct_owner_org_branch` 关联 |
| bank_code | string | 银行编码 | 自定义编码 |
| bank_name | string | 银行中文名称 | 用户说“银行名称 / 开户银行”时取本字段 |
| batch_type | string | 是否支持批扣 | `1` 支持批扣，`0` 不支持批扣 |
| parent | string | 上级银行标志 | 银行层级字段 |
| native_code | string | 本地网标识 | 如需限定本地网需用户确认 |
| status_cd | string | 状态 | 码值含义需确认 |
| payment_method | string | 收费方式 | 用于描述银行支持的收费方式 |

## 常用链路

```sql
select
  ext.acct_owner_org_branch as bank_id,
  bank.bank_name
from tmp_ext_acct ext
left join dws_crm_cfguse.dws_tb_cm_bank bank
  on ext.acct_owner_org_branch = bank.bank_id;
```

## 使用提醒

- 用户说“银行名称 / 开户银行”时，取 `bank_name`，不要把 133.`pay_acct_name` 误当银行名称。
- 是否需要限定 `status_cd`、`native_code` 或银行层级，需按需求确认；默认不猜码值。
