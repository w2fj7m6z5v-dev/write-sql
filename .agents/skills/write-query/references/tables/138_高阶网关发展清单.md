## 常用过滤条件

- 政企全光组网-高阶网关主从网关：`cj_type2 in ('主网关','从网关')`
- 主网关：`cj_type2 = '主网关'`
- 从网关：`cj_type2 = '从网关'`
- 竣工时间：`subs_stat_date`，月范围用 `date_format(subs_stat_date,'yyyyMMdd') between ${yc_date} and ${sum_date}`

## 粒度

服务粒度；同一 `serv_id` 对应一条记录。
