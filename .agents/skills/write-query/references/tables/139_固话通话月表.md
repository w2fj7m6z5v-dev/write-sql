- **业支负责人**: 待确认
- **分区字段**: par_month_id

## 字段说明

| 字段 | 字段类型 | 字段含义 |
|------|---------|---------|
| par_month_id | string | 月份 |
| par_corp_id | string | 本地网（广州固定 = '200'） |
| serv_id | decimal(22,0) | 服务标识 |
| DUR | decimal(22,0) | 通话时长（秒） |

## 常用过滤条件

- 广州固定：`par_corp_id = '200'`
- 通话时长转分钟：`cast(DUR/60 as decimal(22,2))`

## 粒度

服务-月份粒度；同一 `serv_id` 同一 `par_month_id` 一条记录。

## 使用场景

- 固话通话时长查询：按 `serv_id + par_month_id` 关联，取 `DUR` 字段
- 多月份查询时逐月 LEFT JOIN 或按 `par_month_id` 范围聚合后打宽
