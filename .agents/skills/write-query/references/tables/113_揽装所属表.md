- **日表 Hive 表名**：`zone_gz_yz.dwd_yz_sales_man_outlers_final`

## 使用说明

- 本表只包含「有效网点 + 有效揽装人」的对应关系；只有有效网点和有效揽装人才会出现在本表里。
- 查询合同网点下有效销售人员、实际工号数量、有效网点对应揽装人时，优先用本表。
- 统计实际工号数量时优先 `count(distinct staff_id)`；**不要用 `sales_code` 去重或作唯一 JOIN 键**。
- 若网点在本表无记录，不要直接判定网点不存在；应回查 **112 网点维表** 和 **111 揽装人维表** 区分：无效网点、有效网点但无揽装人、有效网点但揽装人无效。
- 从 069 补 **主体编码/主体名称**（`own_operators_nbr`、`own_operators_name`）时，优先按 `channel_nbr` 关联本表；先按 `channel_nbr` 去重，避免一网点多揽装人放大行数。
