# 揽装人维表

- **日表 Hive 表名**: `zone_gz_yz.dwd_yz_sales_man_final`
- **月表 Hive 表名**: `zone_gz_yz.dwd_yz_sales_man_mon_final`
- **业务定位**: 揽装人维表；日表数据唯一，月表按账期快照。
- **分区字段（月表）**: `par_month_id`

---

## 关键字段

| 字段 | 字段类型 | 字段含义 |
|------|---------|---------|
| staff_id |  | 揽装人 / 员工唯一关联键 |
| sales_code |  | 销售编码 / 揽装编码（不唯一） |
| sales_man_nbr |  | 销售员编码 |
| sales_man_name |  | 销售员姓名 |
| sales_man_post_desc |  | 销售员岗位 |
| cert_no |  | 身份证号 |
| status_cd |  | 状态；`S0X` 表示无效 |
| own_channel_id |  | 归属网点 ID |
| par_month_id |  | 月表账期 |

---

## 使用说明

- 需要判断揽装人是否有效时查本表，`status_cd='S0X'` 表示无效。
- 历史账期用月表 `_mon_final`，并按 `par_month_id` 过滤；当前唯一维表可用日表。
- `sales_code` 不唯一，`staff_id` 和 `sales_code` 是 1 对多；关联揽装人只能用 `staff_id`，`sales_code` 只用于展示或辅助过滤。
- 查有效网点下有效揽装人时，优先用 113 揽装所属表；需要诊断无效原因时再回查本表。
