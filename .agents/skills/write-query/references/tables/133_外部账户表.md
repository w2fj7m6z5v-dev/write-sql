# 外部账户表

- **Hive 表名 / 生产现网名**：`dws_crm_cust.dws_ext_acct`
- **用途**：描述电信外部账户相关信息，如银行卡、支付宝、翼支付等；用于回填账户户名、外部账号、开户组织分支机构等。
- **上游入口**：132 支付方案表 `pay_acct_id = ext_acct_id`。
- **默认过滤**：广州固定 `city_id=200`。

## 字段说明

| 字段 | 字段类型 | 字段含义 | 说明 |
| ------ | ------ | ---------- | ------ |
| ext_acct_id | string | 外部账户标识 | 主键；与 132.`pay_acct_id` 关联 |
| cust_id | string | 参与人 / 客户标识 | 外部账户所属客户 |
| pay_channel | string | 银行划扣渠道 | 码值含义需确认 |
| pay_acct_code | string | 外部账户账号 | 敏感字段，输出需确认脱敏 |
| pay_acct_name | string | 外部账户开户名称 / 银行账户名称 | 户名，不是银行名称 |
| acct_owner_org | string | 开户组织分支机构 | 银行或组织相关 ID，含义以生产说明为准 |
| acct_owner_org_branch | string | 开户组织分支机构 / 银行 ID | 常用于关联 134 银行表 `bank_id` |
| status_cd | string | 外部账户状态 | 状态码值需确认 |
| eff_date | string | 生效时间 | 有效性判断候选字段 |
| exp_date | string | 失效时间 | 有效性判断候选字段 |
| ext_acct_type | string | 外部账号类型 | 如银行卡、支付宝、翼支付等类型码 |
| city_id | string | 地市 | 广州固定 `city_id=200` |

## 常用链路

```sql
select
  plan.pay_acct_id,
  ext.pay_acct_name,
  ext.acct_owner_org_branch
from tmp_payment_plan plan
left join dws_crm_cust.dws_ext_acct ext
  on plan.pay_acct_id = ext.ext_acct_id
 and ext.city_id = 200;
```

账户名称脱敏示例：

```sql
case
  when length(ext.pay_acct_name) < 4 then ext.pay_acct_name
  when length(ext.pay_acct_name) = 4 then concat(substr(ext.pay_acct_name, 1, 2), '**')
  when length(ext.pay_acct_name) > 4 then concat(substr(ext.pay_acct_name, 1, length(ext.pay_acct_name) - 4), '****')
  else null
end as pay_acct_name_tm
```

## 使用提醒

- `pay_acct_name` 是外部账户开户名称 / 银行账户名称，也就是户名；**不是银行名称**。
- 需要银行名称 / 开户银行时，继续补 134 银行表，通常按 `acct_owner_org_branch = bank_id` 取 `bank_name`。
- `pay_acct_code`、`pay_acct_name` 属于敏感账户信息，输出时优先脱敏或按用户明确要求处理。
