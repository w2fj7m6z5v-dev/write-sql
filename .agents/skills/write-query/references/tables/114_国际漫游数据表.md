# 国际漫游数据表

- **Hive 表名**：`dws_ctg.dws_mktag_download_share_guoman_label`
- **业务含义**：国际漫游数据表。表内号码为已开通国际漫游权限的号码，可取开户时间、开通国漫权限时间和 IMSI 信息。

---

## 字段说明

| 字段 | 字段类型 | 字段中文名 | 说明 |
|------|------|------------|------|
| latn_id | string | 本地网编码 | 本地网编码 |
| prod_inst_id | string | 产品实例 id | 产品实例 id |
| msisdn | string | 号码 | 号码 |
| reserv1 | string | 用户的开户时间 | 用户的开户时间 |
| reserv2 | string | 开通国漫权限的时间 | 开通国漫权限的时间 |
| reserv3 | string | G_imsi 码 | G_imsi 码 |
| reserv4 | string | L_imsi 码 | L_imsi 码 |
| yyyymmdd | string | 按日分区 | 按日分区 |

---

## 常用条件 / 关联

- 按统计日/分区取数：`yyyymmdd='${day_id}'`。
- 与 069 全业务资料表补字段：`069.acc_nbr = guoman.msisdn`。
- 用户只说“已开通国际漫游号码”时，本表可作为国漫开通事实来源；若还要客户名、局向、产品类型等号码基础属性，先用 069 圈定号码范围后补本表。

## 注意事项

- `yyyymmdd` 是本表日分区/统计日，不是 069 的 `par_month_id`。
- `reserv2` 表示开通国漫权限的时间，不要与用户开户时间 `reserv1` 混用。
- 本表表示国际漫游权限开通，不表示漫游结算收入或实际漫游使用行为。
