## 本表常用条件速查（A 层附注）

| 场景 | 条件片段 |
|------|---------|
| 销售品发展量（含订购+互换） | `action_id IN (1292, 6200) AND COALESCE(subs_stat_reason,'-1') NOT IN ('1200','1300') AND date_format(subs_stat_date,'yyyyMM') IN ('YYYYMM',...)` |
| 是否竣工标记 | `CASE WHEN subs_stat='301200' THEN 1 ELSE 0 END AS is_jg`（默认标记列，不进 WHERE） |
| 排除撤单作废 | `COALESCE(subs_stat_reason,'-1') NOT IN ('1200','1300')` |
| 关联销售品维表 | `LEFT JOIN dws_crm_cfguse.dws_offer f ON a.prod_offer_id = f.offer_id AND f.city_id=200` |
| 关联揽装局向 | `LEFT JOIN dwd_yz_dim_org lz_subst ON a.salestaff_subst_id = lz_subst.org_id AND lz_subst.levs=3`<br>`LEFT JOIN dwd_yz_dim_org lz_branch ON a.salestaff_branch_id = lz_branch.org_id AND lz_branch.levs=4` |

参考 [D_experience/cdap_global_rules.md](../D_experience/cdap_global_rules.md) R-001/R-002/R-003/R-005/R-006。
