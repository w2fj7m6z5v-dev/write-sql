## 本表常用条件速查（A 层附注）

| 叫法/场景 | 条件片段 |
|----------|---------|
| 双线月租 | `yz_cs` |
| 双线速率 | `speed_value`；069 全业务资料表对应账期 `speed_value` 也可取，按查询主路径或用户指定口径选择 |
| 与 069 按账期补双线月租 | `main.acc_nbr = sx.acc_nbr AND main.par_month_id = sx.par_month_id` |

## 使用说明

- 双线范围优先在 069 全业务资料表用 `prod_type2 IN (60,70,71)` 判断：60=互联网专线，70/71=组网专线。
- 本表适合补双线月租 `yz_cs`、双线清单专项字段，以及用户明确指定双线全量清单口径的场景。
- 双线速率 069 和本表均可取；主路径在 069 时可直接取 069 `speed_value`，已补本表时也可使用本表 `speed_value`。
