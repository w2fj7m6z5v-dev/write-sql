## 常用过滤条件

- 政企全光组网-小翼主从网关：`offer_name RLIKE '主网关|从网关'`、`subs_stat = '完工'`
- 主网关：`offer_name like '%主网关%'`
- 从网关：`offer_name like '%从网关%'`
- 竣工时间：`subs_stat_date`，月范围用 `date_format(subs_stat_date,'yyyyMMdd') between ${yc_date} and ${sum_date}`

## 粒度

服务/销售品粒度；同一 `serv_id` 可能对应多条销售品记录。
