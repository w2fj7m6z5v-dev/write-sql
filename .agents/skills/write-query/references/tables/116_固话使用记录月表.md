# 固话使用记录月表

- **Hive 表名**：`summary_ods_month_city.tb_comm_ywl_gw_mon`
- **用途**：按固话号码清单查询指定月份范围内的使用记录 / 使用时长。
- **使用时长口径**：输出分钟 = `dur / 60`。

## 字段说明

| 字段 | 字段类型 | 字段含义 | 说明 |
|------|----------|----------|------|
| par_corp_id | string | 本地网 / 城市分区 | 广州固定 `200` |
| par_month_id | string/int | 统计月份 | 月分区，如 `202603` |
| serv_id | string | 服务标识 | 固话服务标识 |
| acc_nbr | string | 固话号码 | 与附件固话号码匹配 |
| dur | number | 使用时长原始值 | 输出分钟时使用 `dur / 60` |

## 常用条件

| 场景 | 条件 / 处理 |
|------|-------------|
| 广州固话使用记录 | `par_corp_id='200'` |
| 月份范围 | `par_month_id between ${start_month} and ${end_month}` |
| 附件号码清单 | `cast(src.acc_nbr as string) = cast(gw.acc_nbr as string)` |
| 输出使用时长分钟 | `gw.dur / 60 as dur_min` |

## 使用提醒

- 这是固话使用记录事实表，不是固话资料状态、收入或订单动作表。
- 用户要求“到某日”且必须精确到日时，需要确认是否有日级来源；本表按月统计，不能天然表达日内截止。
- 附件驱动输出时保留原始序号；一个号码多个月多行是正常结果。
- 固话号码输出默认脱敏。
