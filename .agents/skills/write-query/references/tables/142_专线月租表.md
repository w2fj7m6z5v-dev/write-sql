## 字段说明

| 字段 | 字段类型 | 字段含义 |
|------|---------|---------|
| par_month_id | string | 月份 |
| serv_id | decimal(22,0) | 服务标识 |
| yz_cs_old | decimal | 月租（旧口径） |

---

## 常用过滤条件

- 按月份：`par_month_id = ${month_id}`

## 粒度

服务-月份粒度；同一 `serv_id` 同一 `par_month_id` 一条记录。

## 使用场景

- 专线月租查询：按 `serv_id + par_month_id` 关联，取 `yz_cs_old` 字段
- 与 033 双线全量清单的区别：033 的 `yz_cs` 是双线全量口径；本表 `yz_cs_old` 是专线月租旧口径，用于尊享专线等场景
