## 1. 需求拆解与澄清

**原始需求**：提取主宽入网量。按字段汇总，字段：月份，分局，营服，揽装人，揽装分局，经营主体，出账收入。

### 1.1 业务澄清 5 问

| 要素 | 内容 |
|------|------|
| 查什么 | 主宽入网量（计数）+ 出账收入（SUM） |
| 查谁 | 主宽带（公众/家庭宽带） |
| 时间 | 2026年5月，按竣工时间 |
| 结果 | 按 月份、分局、营服、揽装人、揽装分局、经营主体 分组汇总 |
| 限制 | 排除团单/商务等特定低档套餐？→ 标准指标口径 `kd_desc='普通宽带'` |

### 1.2 关键确认

| 确认项 | 用户回答 | 影响 |
|--------|---------|------|
| 时间范围 | 202605 月份 | 锁定分区 `par_month_id='202605'` |
| 入网口径 | 按竣工时间（`open_date`） | 使用 `date_format(open_date,'yyyyMM')='202605'` |
| 出账收入 | 入网当月收入（`fee` 公允后） | 使用 `SUM(fee)`，非税后或其他口径 |
| 竣工时间字段 | 使用 `open_date`（服务开通时间） | 与标准指标 M-BASIC-BB-001 一致 |
| 出账收入字段 | `fee`（出账收入公允后） | 确认口径 |

```sql
-- 主宽入网数标准口径 SQL
SELECT COUNT(serv_id) AS main_broadband_new_user_cnt
FROM dwm_yz_tb_comm_cm_all_final
WHERE par_month_id = ${month_id}
  AND kd_desc = '普通宽带'
  AND is_new_user = 1
  AND date_format(open_date, 'yyyyMM') = '${month_id}'
  AND prod_type = 40
;
```

## 3. 字段映射

| 需求字段 | 来源表 | 来源字段 | 是否补表 |
|----------|--------|---------|---------|
| 月份 | 069 自带 | `par_month_id` | ❌ 不补 |
| 分局 | 069 自带 | `subst_name`（划小分局名称） | ❌ 不补 |
| 营服 | 069 自带 | `branch_name`（划小营服名称） | ❌ 不补 |
| 揽装人 | 069 自带 | `sales_name` | ❌ 不补 |
| 揽装分局 | 069 自带 | `channel_subst_name`（网点所属区县） | ❌ 不补 |
| 经营主体 | 113 揽装所属表 | `own_operators_name` | ✅ 需补表 |
| 出账收入 | 069 自带 | `SUM(fee)` | ❌ 不补 |

### 3.1 字段语义确认

- **分局 vs 揽装分局**：
  - `subst_name` = 划小分局（号码/服务归属的分局）
  - `channel_subst_name` = 网点所属区县（揽装人所在网点归属的分局）
  - 两者是不同业务概念，均为 069 自带字段，无需补表

## 4. 补表方案

### 4.1 经营主体（唯一缺口字段）

**初始方案**（已纠正）：~~补 112 网点维表~~（FIELD_BACKFILL.md 推荐）

**纠正方案**：补 **113 揽装所属表**

**纠正原因**：069 表文档明确记载"固定补 113 揽装所属表"，按冲突优先级表文档(第3级) > FIELD_BACKFILL(第4级)。

### 4.2 补表详情

| 缺口字段 | 补表 | Hive 表名 | JOIN 键 | 去重方式 | 行数风险 |
|----------|------|-----------|---------|----------|---------|
| 经营主体 | 113 揽装所属表 | `zone_gz_yz.dwd_yz_sales_man_outlers_final` | `069.channel_nbr = 113.channel_nbr` | `ROW_NUMBER() OVER (PARTITION BY channel_nbr ORDER BY staff_id) = 1` | 一网点多揽装人，去重后一对一，无放大 |

### 4.3 补表自检

113 表结构核心字段：
- `channel_nbr` → 与 069 的 `channel_nbr` 关联
- `own_operators_nbr` → 经营主体编码
- `own_operators_name` → 经营主体名称
- 113 表只含「有效网点 × 有效揽装人」记录

## 5. 最终 SQL

```sql
-- ============================================================
-- 主宽入网量按字段汇总
-- 账期: 2026年5月
-- 口径: 按竣工时间（open_date）= 入网当月
-- 主表: dwm_yz_tb_comm_cm_all_final (069 全业务资料表)
-- 补表: zone_gz_yz.dwd_yz_sales_man_outlers_final (113 揽装所属表)
-- ============================================================

WITH dim_channel AS (
    -- 113 揽装所属表：按 channel_nbr 去重（一网点多揽装人），取经营主体
    SELECT channel_nbr,
           own_operators_nbr,
           own_operators_name
    FROM (
        SELECT channel_nbr,
               own_operators_nbr,
               own_operators_name,
               ROW_NUMBER() OVER (PARTITION BY channel_nbr ORDER BY staff_id) AS rn
        FROM zone_gz_yz.dwd_yz_sales_man_outlers_final
    ) t
    WHERE rn = 1
)
SELECT a.par_month_id                            AS 月份,
       a.subst_name                              AS 分局,
       a.branch_name                             AS 营服,
       a.sales_name                              AS 揽装人,
       a.channel_subst_name                      AS 揽装分局,
       b.own_operators_name                      AS 经营主体,
       COUNT(a.serv_id)                          AS 入网量,
       ROUND(SUM(a.fee), 2)                      AS 出账收入
FROM dwm_yz_tb_comm_cm_all_final a
LEFT JOIN dim_channel b
    ON a.channel_nbr = b.channel_nbr
WHERE a.par_month_id = '202605'
  AND a.is_new_user = 1
  AND a.prod_type = 40
  AND a.kd_desc = '普通宽带'
  AND date_format(a.open_date, 'yyyyMM') = '202605'
GROUP BY a.par_month_id,
         a.subst_name,
         a.branch_name,
         a.sales_name,
         a.channel_subst_name,
         b.own_operators_name
;
```
