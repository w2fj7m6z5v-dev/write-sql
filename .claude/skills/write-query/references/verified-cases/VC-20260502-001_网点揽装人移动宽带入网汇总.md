# 网点/揽装人维度移动宽带入网汇总

来源：`CDAP自助分析常用统计语句分享.docx`

## 适用

- 用户要按网点、揽装人统计移动入网、宽带入网、融合主卡、副卡、千兆、专线等多个同源指标。
- 用户已接受普通入网/到达默认走 069 全业务资料表。

## 不适用

- 用户明确要求某个专项清单字段。
- 用户要求标准指标且指标文件已有更严格口径。

## 主表与口径

- 主表：069 全业务资料表，生产表 `dwm_yz_tb_comm_cm_all_final`。
- 分区：`par_month_id=${month_id}`。
- 入网业务月：`date_format(open_date,'yyyyMM')='${month_id}'`。

## 字段映射

| 需求字段 | 来源 |
|----------|------|
| 网点编码 | 069.`channel_nbr` |
| 网点名称 | 069.`channel_name` |
| 网点子类型 | 069.`channel_type` |
| 主体编码 | 113.`own_operators_nbr`，按 `channel_nbr` 补 |
| 主体名称 | 113.`own_operators_name`，按 `channel_nbr` 补 |
| 揽装人工号 | 069.`sales_code` |
| 揽装人 | 069.`sales_name` |

## 补表

如果需要主体编码/主体名称，固定补 113 揽装所属表；先按 `channel_nbr` 去重，避免一网点多人员放大行数。其它字段默认不补表。

## SQL 编排

```sql
SELECT
  channel_nbr,
  channel_name,
  channel_type,
  sales_code,
  sales_name,
  COUNT(DISTINCT CASE WHEN is_new_user=1 AND date_format(open_date,'yyyyMM')='${month_id}' AND prod_type=30 THEN serv_id END) AS mv_new_cnt,
  COUNT(DISTINCT CASE WHEN is_new_user=1 AND date_format(open_date,'yyyyMM')='${month_id}' AND prod_type=30 AND yd_prod_type1='预付费单产品' THEN serv_id END) AS mv_prepaid_cnt,
  COUNT(DISTINCT CASE WHEN is_new_user=1 AND date_format(open_date,'yyyyMM')='${month_id}' AND prod_type=30 AND yd_prod_type1='后付费单产品' THEN serv_id END) AS mv_postpaid_cnt,
  COUNT(DISTINCT CASE WHEN is_new_user=1 AND date_format(open_date,'yyyyMM')='${month_id}' AND prod_type=30 AND yd_prod_type1='融合' THEN serv_id END) AS mv_rh_main_cnt,
  COUNT(DISTINCT CASE WHEN is_new_user=1 AND date_format(open_date,'yyyyMM')='${month_id}' AND prod_type=30 AND yd_prod_type1='副卡' THEN serv_id END) AS mv_vice_cnt,
  COUNT(DISTINCT CASE WHEN is_new_user=1 AND date_format(open_date,'yyyyMM')='${month_id}' AND prod_type=40 AND kd_desc='普通宽带' THEN serv_id END) AS bb_main_new_cnt,
  COUNT(DISTINCT CASE WHEN is_new_user=1 AND date_format(open_date,'yyyyMM')='${month_id}' AND prod_type=40 AND kd_desc='普通宽带' AND is_rh_ykj>0 THEN serv_id END) AS bb_rh_new_cnt,
  COUNT(DISTINCT CASE WHEN is_new_user=1 AND date_format(open_date,'yyyyMM')='${month_id}' AND prod_type=40 AND kd_desc='普通宽带' AND speed_value>=1000 THEN serv_id END) AS bb_gigabit_new_cnt,
  COUNT(DISTINCT CASE WHEN is_new_user=1 AND date_format(open_date,'yyyyMM')='${month_id}' AND prod_type2 IN (60,70) THEN serv_id END) AS line_new_cnt
FROM dwm_yz_tb_comm_cm_all_final
WHERE par_month_id=${month_id}
  AND subst_name='${subst_name}'
  AND is_new_user=1
GROUP BY channel_nbr, channel_name, channel_type, sales_code, sales_name
;
```
