# 固话使用记录

## 适用

- 用户给固话号码清单，要求查询指定月份范围内的使用记录或使用时长。
- 需要输出使用时长分钟。

## 不适用

- 固话资料状态、收入、订单动作：不要用 116。
- 用户要求严格截止到某日且没有日级来源：不能只靠月表默认满足，需确认是否有日表。

## 主表与口径

| 角色 | 表 | 用途 |
|---|---|---|
| 驱动表 | 用户附件固话号码清单 | 保留原始序号和号码 |
| 使用记录 | 116 固话使用记录月表 `summary_ods_month_city.tb_comm_ywl_gw_mon` | 按号码和月份取使用时长 |

## 关键规则

- 广州固定 `par_corp_id='200'`。
- 按 `par_month_id` 限定月份范围。
- 附件号码按 `acc_nbr` 匹配，必要时统一转字符。
- 使用时长输出分钟：`dur / 60`。
- 一个号码跨多个月输出多行是正常结果。
- 固话号码输出默认脱敏，除非用户明确要求明文。

## 输出字段建议

| 需求字段 | 来源 |
|---|---|
| 原始序号 / 固话号码 | 用户附件 |
| 月份 | 116 `par_month_id` |
| 使用时长秒 | 116 `dur` |
| 使用时长分钟 | `dur / 60` |

## 自检

- 附件输入行数。
- 命中号码数。
- 输出明细行数。
- 各月份分布，确认月份范围没有漏取。
