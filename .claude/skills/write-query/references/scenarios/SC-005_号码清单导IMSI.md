# 号码清单导 IMSI

## 适用

- 用户给普通号码清单，要求导出号码对应 IMSI。
- 号码 IMSI 按 105 产品规格属性处理。

## 不适用

- 国际漫游权限场景下的 G/L IMSI：用 `SC-002_国际漫游开通权限.md`。
- 附属产品属性：用 106 附属产品资料表，不走本场景。

## 主表与补表

| 角色 | 表 | 用途 |
|---|---|---|
| 驱动表 | 用户附件号码清单 | 保留原始序号和号码 |
| 主表 | 069 全业务资料表 | 按号码和账期定位 `serv_id` |
| 补表 | 105 特性资料表 | `attr_id='200000103'` 输出 `attr_value1` |

## 关键规则

- 未给号码在网账期时，先确认 `par_month_id`。
- 先按 `附件号码 = 069.acc_nbr` 且 `069.par_month_id=${month_id}` 定位 `serv_id`。
- 再补 105：`069.serv_id = attr.serv_id`，`attr.attr_id='200000103'`。
- 当前在网可用 105 日表；历史账期必须用 105 月表 `iodata_ods_month_city.tb_pre_cm_attr_all_mon`，并按 `par_month_id` 与 069 对齐。
- 不使用 `dws_crm_cust.dws_prod_inst_attr`；本知识库不把它作为独立补表路径。
- 普通 IMSI 不要默认走 114 国际漫游数据表。

## 输出字段建议

| 需求字段 | 来源 |
|---|---|
| 原始号码 / 序号 | 用户附件 |
| 服务标识 | 069 `serv_id` |
| IMSI | 105 `attr_value1` |

## 自检

- 附件输入行数。
- 069 命中行数。
- 105 命中行数。
- IMSI 空值率。
