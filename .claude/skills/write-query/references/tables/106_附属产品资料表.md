# 附属产品资料表（附属产品属性）

- **Hive 表名（日表 / 在网）**：`summary_ods_day_city.rpt_comm_cm_subserv` — 当前在网号码附属产品特性
- **Hive 表名（月表 / 历史）**：`iodata_ods_month_city.rpt_comm_cm_subserv_mon` — 历史月份附属产品特性快照

> ⚠️ **勿与 105 产品规格属性混用**：产品规格走 `tb_pre_cm_attr_all(_mon)`；附属产品走本表。日表只在网；历史/拆机前月必须用月表。

---

## 与 105 特性资料表对照

| 维度 | 105 产品规格属性 | 106 附属产品属性（本表） |
|------|------------------|--------------------------|
| 业务含义 | 主产品规格特性 | 附属产品的特性 |
| 月表 | `tb_pre_cm_attr_all_mon` | `rpt_comm_cm_subserv_mon` |
| 关联键 | `serv_id` + `attr_id` | `serv_id` + `attr_id` |
| 额外字段 | — | `sub_prod_id`（附属产品 ID，可输出） |
| 分区 | `par_corp_id='200'`，`par_month_id` | 同左 |

## 字段说明

| 字段 | 字段类型 | 字段含义 | 说明 |
| ------ | ------ | --------- | ------ |
| serv_id | decimal(22,0) | 服务标识 | 关联 069、种子表等 |
| sub_prod_id | decimal(22,0) | 附属产品 ID | 可带出；JOIN 默认不需限定（`serv_id+attr_id` 业务唯一） |

| create_date |  | 订购时间 | 特性订购时间 |
| state_date | timestamp | 状态时间 | 语义以生产 `DESC` 为准 |
| par_corp_id | decimal(10,0) | 城市/公司分区 | 广州固定 `200` |
| par_month_id |  | 统计月份（月表） | 属性快照账期 |

## 本表常用条件速查

| 场景 | 条件 |
|------|------|
| 广州附属产品特性 | `par_corp_id='200'` |
| 历史某月快照 | 月表 `par_month_id=${attr_month_id}` |
| 指定附属产品属性 | `attr_id IN (...)` |
| 指定附属产品名称 / ID | 先用 130 附属产品配置表按 `seq_id=12 AND seq_type='sub_prod_id'` 圈 `seq_value_id`，再关联 `sub_prod_id` |
| 拆机前一月特性 | 先 069 月表定 `cancel_month_id`，再 `attr_month_id = cancel_month_id - 1` |

## 特性值中文名

补 `dws_crm_cfguse.dws_attr_value`（`tables/015_字典表视图.md`）：

```sql
a.attr_id = b.attr_id
AND a.attr_value1 = b.attr_inner_value
AND b.city_id = '200'
```

---

## 特性名称 / 附属产品名称

- 特性名称补 `dws_crm_cfguse.dws_attr_spec`（`tables/016_字典维表视图.md`）：

```sql
a.attr_id = spec.attr_id
-- 取 spec.attr_name
```

- 附属产品名称、附属产品配置编码补 `dwd_dim_all_config`（`tables/130_附属产品配置表.md`）：

```sql
a.sub_prod_id = cfg.seq_value_id
AND cfg.seq_id = 12
AND cfg.seq_type = 'sub_prod_id'
-- 取 cfg.seq_name / cfg.seq_value_code
```

若需求只给附属产品编码或产品规格编码，可先用 016 的 `attr_inner_cd` 反查 `attr_id, attr_name`，再回本表取附属产品特性值。

## 关联说明

| 关联对象 | JOIN 键 | 备注 |
|----------|---------|------|
| 069 全业务资料表 | `serv_id` | 拆机月定位用 069 **月表** |
| 种子表 | `serv_id` | 种子驱动 LEFT JOIN |
| 字典表 | `attr_id` + `attr_value1=attr_inner_value` | 输出中文名 |
| 特性规格维表 | `attr_id` | 输出 `attr_name`；也可按 `attr_inner_cd` 反查 `attr_id` |
| 附属产品配置表 | `sub_prod_id = seq_value_id` | 按附属产品名称 / 编码圈定或输出配置名称 |

## 注意事项

- 当前约定：`serv_id + attr_id` 在月表给定账期内业务唯一；若出现多行，按 `create_date`/`state_date` 取最新或加 `sub_prod_id` 限定。
- 不要用日表取已拆机号码的历史附属产品特性。
- 详见 `verified-cases/VC-20260522-001_种子serv_id拆机前月产品规格属性宽表.md`。
