# 产权客户全量表

## 用途

用于查询产权客户全量信息。用户提供产权客户名称但缺少产权客户编码时，可用本表兜底匹配产权客户编码。

适用范围是客户实体映射或客户信息维护，例如按客户名称补产权客户编码、更新客户资料。若需求是号码/服务明细，并希望输出号码对应的产权客户名或客户编码，优先使用 069 或当前事实主表自带客户字段。

## 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| cust_name |  | 产权客户名称 |
| cust_number |  | 产权客户编码 |

## 常用关联

```sql
-- 附件缺产权客户编码时，按客户名称兜底匹配产权客户编码
left join dws_crm_cust.dws_customer c
  on seed.cust_name = c.cust_name
```

## 注意

- 优先使用产权客户编码精确匹配；客户名匹配只作为无编码兜底。
- 客户名可能重名或多命中，回填时必须保留附件序号/原始行标识，并提示多匹配风险。
