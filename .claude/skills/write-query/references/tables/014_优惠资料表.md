# 优惠资料表

- **Hive 表名**: `ads_yz_rpt_comm_cm_msdisc_final`
- **月表 / 历史快照表名**: `dwd_yz_rpt_comm_cm_msdisc_mon_final`
- **视图名**: view_ads_yz_rpt_comm_cm_msdisc_final
- **业务负责人**: 
- **业支负责人**: 杨洋
- **分区字段**: par_month_id

---

## 使用说明

- 查销售品是否在档、销售品到期时间、套餐实例关系时，本表是存量事实表；订单订购 / 发展动作不要改走本表，走 041 优惠订单表。
- 需要历史账期快照、历史协议期、历史 `limit_date` 回溯时，用月表 `dwd_yz_rpt_comm_cm_msdisc_mon_final` 并锁 `par_month_id`。
- 协议期 / 合同年限核查可从本表取 `open_date`、`limit_date`、`prod_offer_id`，但具体协议销售品范围和年限系数应来自确认后的配置表或业务口径，不从案例号码硬编码。
