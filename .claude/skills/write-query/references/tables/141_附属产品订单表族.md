# 附属产品订单表族

> ⚠️ **与 106 附属产品资料表的区别**：106 = 附属产品**属性**快照（`serv_id + attr_id`，无订单流水）；本表族 = 附属产品**订单**流水（有 `oper_type`、`subs_stat`、`subs_code`、`subs_stat_date` 等订单字段）。需要查附属产品的开通/取消/变更记录时用本表族，查当前附属产品属性值时用 106。

- **Hive 表名（日-当前）**：`summary_ods_day_city.rpt_comm_ba_sub_prod` — 当前在网附属产品订单
- **Hive 表名（日-历史）**：`summary_ods_day_city.rpt_comm_ba_sub_prod_hist` — 历史附属产品订单（含已取消）
- **Hive 表名（月-当前）**：`iodata_ods_month_city.rpt_comm_ba_sub_prod_mon` — 月表当前快照
- **Hive 表名（月-历史）**：`iodata_ods_month_city.rpt_comm_ba_sub_prod_hist_mon` — 月表历史快照

---

## 字段说明

| 字段 | 字段类型 | 字段含义 | 说明 |
| ------ | ------ | --------- | ------ |
| serv_id | decimal(22,0) | 服务标识 | 关联 069、种子表等 |
| acc_nbr | varchar(96) | 接入号 |  |
| sub_prod_id | decimal(22,0) | 附属产品 ID | 如 600031007=全网通 |
| prod_id | decimal(22,0) | 产品 ID |  |
| oper_type | varchar(64) | 操作类型 | 1000=开通，1100=取消 |
| subs_stat | varchar(64) | 订单状态 | 301200=竣工 |
| subs_stat_reason | varchar(64) | 订单状态原因 |  |
| subs_code | varchar(64) | 订单编码 |  |
| subs_stat_date | timestamp | 订单状态时间 | 竣工/取消时间 |
| par_corp_id | decimal(10,0) | 城市/公司分区 | 广州固定 `200` |
| par_month_id |  | 统计月份（月表） | 月表快照账期 |

---

## 常用过滤条件

| 场景 | 条件 |
|------|------|
| 开通记录 | `oper_type = '1000'` |
| 取消记录 | `oper_type = '1100'` |
| 竣工订单 | `subs_stat = '301200'` |
| 排除撤单/作废 | `subs_stat_reason not in ('1200','1300')` |
| 指定附属产品 | `sub_prod_id = ${sub_prod_id}` |
| 广州 | `par_corp_id = '200'` |

---

## 四表使用场景

| 表 | 场景 | 说明 |
|------|------|------|
| `rpt_comm_ba_sub_prod` | 查当前在网的附属产品订单 | 日表，只在网 |
| `rpt_comm_ba_sub_prod_hist` | 查历史附属产品订单（含已取消/已拆机） | 日表，覆盖历史 |
| `rpt_comm_ba_sub_prod_mon` | 按月查在网附属产品订单快照 | 月表 `par_month_id` |
| `rpt_comm_ba_sub_prod_hist_mon` | 按月查历史附属产品订单快照 | 月表 `par_month_id` |

> **典型用法**：查某附属产品的开通/取消历史 → UNION ALL 四张表去重，按 `serv_id` 聚合取最新 `oper_type` 和时间。

## 关联说明

| 关联对象 | JOIN 键 | 备注 |
|----------|---------|------|
| 069 全业务资料表 | `serv_id` | 补号码状态、资料 |
| 种子表 | `serv_id` | 种子驱动 LEFT JOIN |
| 106 附属产品资料表 | `serv_id` + `sub_prod_id` | 订单+属性联动 |

---

## 注意事项

- 查历史订单（含已拆机）时，需要 UNION ALL 日-历史 + 月-历史，仅查日-当前会漏已拆机号码记录。
- `oper_type` 为码值，不要用中文匹配。
- 同一 `serv_id + sub_prod_id` 可能有多条订单记录，按 `subs_stat_date` 取最新。
