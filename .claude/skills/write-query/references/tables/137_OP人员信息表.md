## 使用说明

- 当前月与历史月数据合并在一张表，按 `par_month_id` 分区区分。
- 查询特定月份数据时需加 `par_month_id` 过滤。
- 用于 531 薪酬相关统计，如各薪酬岗位人数查询。
- **薪酬岗位字段**：`xc_staff_lx`（人员类型-薪酬），用于按薪酬岗位分组统计。
- **分公司过滤**：用 `subst_name`（三级机构/县分名称），如 `subst_name = '天河分公司'`。
- 人员类型标记（`is_qdzkhjl` ~ `is_yyy`）按优先级 1-6 判定，`staff_lx` 为最终分类结果。
