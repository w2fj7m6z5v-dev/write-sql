# 未出账明细表

- **Hive 表名**: `dws_acct.dws_unacct_item`
- **业务负责人**: 待确认
- **业支负责人**: 待确认

---

## 字段说明

| 字段 | 字段类型 | 字段含义 |
|------|---------|---------|
| billing_cycle_id | string/int | 账期 |
| prod_inst_id | decimal(22,0) | 产品实例 ID，等于 serv_id |
| amount | decimal | 未出账金额 |
| acct_item_type_id | string | 科目类型 ID |
| city_id | string | 城市 ID |

---

## 常用过滤条件

- 按城市：`city_id = '200'`（广州）
- 按科目类型：`acct_item_type_id IN (...)`（时长类非费用项判断使用记录）

## 粒度

账期-产品实例-科目明细粒度；同一 `prod_inst_id` 同一 `billing_cycle_id` 可能多行（不同科目类型）。

## 使用场景

- 使用记录查询：按 `serv_id + billing_cycle_id` 关联，通过时长类非费用项 `acct_item_type_id` 判断是否有使用行为
- 欠费追缴：查某客户所有业务每月的使用记录，有使用按套餐追缴、无使用免除月租
- 与 116 固话使用记录表的区别：116 仅覆盖固话通话时长；本表覆盖全产品未出账使用明细

## 关联说明

| 关联对象 | JOIN 键 | 备注 |
|----------|---------|------|
| 069 全业务资料表 | `prod_inst_id = serv_id` | 补号码/服务资料 |
| 种子表 | `prod_inst_id = serv_id` | 种子驱动 LEFT JOIN |

## 注意事项

- `acct_item_type_id` 码值列表由省公司指引《时长类非费用项判断是否有使用记录》定义，不同场景可能不同，不硬编码
- 按 `billing_cycle_id + prod_inst_id` 聚合 `sum(amount)` 得到每服务每月使用金额
- 同一 `prod_inst_id` 同一 `billing_cycle_id` 可能有多条科目明细，聚合前先确认去重口径
